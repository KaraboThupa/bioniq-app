# ISP App

Modern ISP client management application with AI-powered support and network tools.

---

# Overview

ISP App is a mobile application prototype designed for Internet Service Providers (ISPs) and their customers.

The platform focuses on improving customer experience by combining:
- network management
- billing visibility
- device control
- AI-powered support

The goal is to reduce manual support workload while giving customers direct control over their internet services.

# Features

- Secure login using customer ID
- Wi-Fi management
- Connected device monitoring
- Disconnect or block unknown devices
- Billing and payment visibility
- Service restriction for overdue accounts
- AI-powered troubleshooting assistant
- Customer self-service tools

# Project Vision

This project explores the concept of a modern self-service ISP ecosystem where customers can:
- manage their own network
- troubleshoot issues instantly
- reduce dependency on manual support
- interact with AI-powered assistance

The AI assistant acts as the first line of support before escalation to technicians.
